OCaml-RSS

What is OCaml-RSS ?

OCaml-RSS is a small OCaml library providing functions to parse and print RSS 2.0 files. The parser can also parse some RDF files, but some fields are not taken into account.

OCaml-RSS was previously part of Cameleon but is now developped separately and is findlib compatible.

Download

OCaml-RSS is hosted on Github.

It is also available in OPAM.

Releases

  • 2.2.1 [2013-06-07]:
    • fix: do not set default namespace in final rss,
    • fix: rdf parsing,
    • fix: Rss.sort_items_by_date sorts with old items last, as documented.
  • 2.2.0 [2013-04-15]:
    • add Rss.compare_item,
    • fix: provide polymorphic functions on channel_t and on item_t,
    • add Rss.channel_t_of_xmls,
    • add Rss.xml_of_source,
    • fix #5: incorrect sort by date.
  • 2.1.0 [2013-03-19]:
    • handle namespaces (the list is stored in the channel_t structure),
    • allow the library user to read addition information from <item> and <channel> prefixed subnodes, as the RSS 2.0 specification indicates this is the way to extend RSS information,
    • handle "cloud", "skipHours", "skipDays" and "rating" in channels,
    • use Neturl.url instead of string to represent URLs,
    • use Netdate instead of Rss_date, which was a copy of Netdate,
    • OCamlRSS now depends on OCamlnet,
    • default encoding is now UTF-8,
    • new parameter "indent" for printing,
    • new "opts" parameter when reading,
    • reading returns error list besides channel.
    Beware that this release introduces backward-incompatible changes.
  • 2.0 [2012-04-06]: first release after extraction from Cameleon. Now uses Xmlm instead of Xml-light.

Installation

OCaml-RSS requires Xmlm >= 1.1.0 and OCamlnet >= 3.6.

To compile and install:

  • tar xvfz ocamlrss-XXX.tar.gz
  • cd ocamlrss-XXX
  • make all
  • make install
